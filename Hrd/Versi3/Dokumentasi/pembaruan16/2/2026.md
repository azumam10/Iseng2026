📌 Instalasi Role & Permission Management
composer require spatie/laravel-permission

Keterangan:

Perintah di atas digunakan untuk menginstall package spatie/laravel-permission, yaitu library tambahan pada framework Laravel yang berfungsi untuk mengelola Role (peran) dan Permission (hak akses) pada sistem aplikasi.

Package ini dikembangkan oleh Spatie dan umum digunakan dalam pengembangan aplikasi berbasis Laravel untuk kebutuhan otorisasi yang terstruktur.

🎯 Tujuan Penggunaan

Package ini digunakan untuk:

Membuat dan mengelola Role (contoh: Admin, HR, Manager, Employee).

Membuat dan mengatur Permission (contoh: tambah data, edit data, approve cuti).

Menghubungkan Role dengan Permission.

Memberikan Role kepada User.

Melakukan pembatasan akses fitur berdasarkan Role atau Permission.

📂 Implementasi Lanjutan Setelah Instalasi

Setelah menjalankan perintah instalasi, langkah berikutnya adalah:

1️⃣ Publish konfigurasi dan migration
php artisan vendor:publish --provider="Spatie\Permission\PermissionServiceProvider"

Perintah ini akan:

Membuat file konfigurasi config/permission.php

Membuat file migration untuk tabel role dan permission

2️⃣ Menjalankan migration
php artisan migrate

Perintah ini akan membuat tabel berikut pada database:

roles

permissions

model_has_roles

model_has_permissions

role_has_permissions

📌 Instalasi Tailwind CSS dan DaisyUI
npm install -D tailwindcss postcss autoprefixer daisyui
🎯 Tujuan

Perintah ini digunakan untuk menginstall dependensi frontend yang dibutuhkan untuk styling aplikasi berbasis CSS modern, khususnya pada project Laravel yang menggunakan Vite atau build tool berbasis Node.js.

📦 Penjelasan Package yang Diinstall

tailwindcss
Framework CSS berbasis utility-first untuk mempercepat pembuatan tampilan antarmuka (UI).

postcss
Tool untuk memproses CSS dengan plugin tambahan (termasuk Tailwind).

autoprefixer
Plugin PostCSS yang otomatis menambahkan vendor prefix CSS agar kompatibel dengan berbagai browser.

daisyui
Plugin komponen UI berbasis Tailwind CSS yang menyediakan komponen siap pakai seperti:

Button

Card

Modal

Navbar

Form

Alert

Instalasi menggunakan flag:

-D

yang berarti dependency hanya digunakan saat development (devDependencies) dan tidak dibutuhkan pada environment production server.

📌 Inisialisasi Konfigurasi Tailwind
npx tailwindcss init -p
🎯 Tujuan

Perintah ini digunakan untuk membuat file konfigurasi awal Tailwind CSS.

Perintah tersebut akan menghasilkan dua file:

tailwind.config.js
Berfungsi untuk mengatur:

Path file yang dipindai Tailwind

Theme customization

Plugin tambahan (misalnya DaisyUI)

postcss.config.js
Digunakan untuk mengkonfigurasi PostCSS agar dapat memproses Tailwind dan Autoprefixer.
